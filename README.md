# Medicine Schedule Tracker

A C# console application to manage daily medicine schedules with full 
CRUD operations — add medicines, view daily schedule by time slot, 
mark doses as taken, and remove entries.

Also includes a relational SQL schema (queries.sql) designed to back 
a database-driven version of the same application.

**Tech:** C#, .NET, SQL (MySQL)  
**Author:** Saswat Mohallick
